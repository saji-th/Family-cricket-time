# Hansili Ayata Criket - Live Stream

This is a premium live stream player website.

## How to use
1. Open `index.html` in your web browser.
2. The stream should load automatically.

## How to host on GitHub (since Git is not installed)
1. Go to [GitHub.com](https://github.com) and sign in.
2. Click the `+` icon in the top right and select "New repository".
3. Name your repository (e.g., `cricket-live-stream`).
4. Click "Create repository".
5. On the next page, look for the link "uploading an existing file".
6. Drag and drop `index.html` and `style.css` from this folder into the browser window.
7. Click "Commit changes".
8. To make it a live website:
   - Go to the repository **Settings**.
   - Click on **Pages** in the left sidebar.
   - Under **Source**, select `main` branch.
   - Click **Save**.
   - Wait a few minutes, and your site will be live at the provided URL!

## Customization
- You can edit `index.html` to change the channel ID if needed.
- You can edit `style.css` to change colors and layout.
